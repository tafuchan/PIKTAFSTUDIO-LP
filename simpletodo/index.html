About

Simple ToDo について

日々のタスク管理を「シンプル・確実・自分好み」にこなすための ToDo アプリです。
「やることリスト」「習慣チェック」「お買い物メモ」など、用途に合わせて自由に使えます。
煩雑な機能を省き、必要なものだけを丁寧に作りました。

For

こんな方におすすめ

  • ToDo アプリは色々試したけど、ごちゃごちゃして続かなかった
  • 習慣を作りたいけど、市販のアプリは多機能すぎる
  • お買い物メモや日課のチェックを、紙のように使いたい
  • ウィジェットで、ホーム画面から直接タスクをチェックしたい

Features

主な機能

01

直感的なタスク管理

ワンタップで追加・完了・削除。長押しで複数選択し、一括移動・削除・繰り返し設定変更が可能。カテゴリタブで分類、ドラッグで自由に並び替え。

02

繰り返しタスクの自動リセット

毎日 / 曜日指定 / 毎月 X 日から選択。リセット時刻を自分で設定（例: 朝5時にリセット）。指定時刻に自動で未完了に戻り、習慣化をサポート。

03

ホーム画面ウィジェット

24色のプリセット + 完全カスタムカラー対応。背景透明度・コンテンツ背景色の指定、ヘッダー分離など4種類のデザインスタイル。ウィジェットから直接タスク完了・追加も可能。

04

PDF 出力

シンプルリスト形式やチェックリスト表形式で出力。日付列付きで、紙のチェック表として印刷可能。カテゴリ別の選択出力にも対応。

05

バックアップ・復元

Google ドライブ等、お好きな場所に JSON 形式でバックアップ。当方サーバーへの自動送信は一切なし。機種変更時もスムーズにデータ移行できます。

06

細かい使い心地

ダークモード対応。テーマカラー20色から選択。日本語 / 英語切替対応。完了タスクの一括削除、非表示フィルタも備えています。

07

広告非表示オプション

一度の購入（¥360）で広告を完全に非表示にできます。機能制限はなく、すべての機能は無料版でも利用可能です。

08

アカウント登録不要

インストールしたらすぐに使えます。メールアドレスやアカウントの登録は一切不要。「使い始める」ハードルを徹底的に下げました。

Privacy

プライバシーへの配慮

データはあなたの端末の中だけ

  • タスクデータは 端末内のローカルストレージ（Room / DataStore） にのみ保存
  • 当方サーバーへの自動送信は一切ありません
  • バックアップ先（Google ドライブ等）はあなたが自由に選択できます
  • アカウント登録不要 — 個人情報の取得はありません
